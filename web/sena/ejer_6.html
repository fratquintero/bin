Programación de algoritmos en JavaScript

Enunciado del problema

Una temperatura Celsius (centígrados) puede ser convertida a una temperatura equivalente Fahrenheit de acuerdo con la siguiente fórmula: F = (9/5)C + 32 




function ejercicioSeis() {
      let tempC = document.getElementById("num1").value;
      let tempF = 9/5*tempC + 32;
      document.getElementById("result").innerHTML ="Temperatura en grados Fahrenheit es igual a "+tempF;     
}
