Programación de algoritmos en JavaScript

Enunciado del problema

Codifique en JavaScript un algoritmo que sume los números pares comprendidos entre 50 y 200, inclusive 


function ejercicioCinco() {
      let number = 50;
      let sum = 0;
      while(true) {
            if (number > 200) break;
            sum += number;
            number += 2;    
      }
     document.getElementById("result").innerHTML ="La suma es igual a "+sum; 
}
